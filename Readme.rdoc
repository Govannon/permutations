= Permutations

Generate permutations for nested arrays or strings that offer one or more choices.
You can use it as a Rails plugin.

== Example

  [[1,2],[3,4]].permutations # =>[[1,3],[1,4],[2,3],[2,4]]
  "{1,2}{3,4}".permutations # => ["13", "14", "23", "24"]
